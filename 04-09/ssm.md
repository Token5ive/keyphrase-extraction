# Yesterday (어제 한 일)
- 모델 개선: 다양한 인코더 모델 변경을 통한 성능 비교 실험 추가 수행
- 코드 리팩토링: 실험 코드 및 주요 로직의 모듈화 진행 (유지보수성 향상)
- 평가 지표 추가 (ILD, SR)

# Today (오늘 할 일)
- 추론 End-to-End: 잔여 실험 로직 모듈화 완료 및 코드 통합(다음 과정 모두 완료후 진행)
- 실험 결과 분석: 지표 해석 및 성능 차이에 대한 가설 설정/검증 -> 문서화
- 파이프라인 검증: 모듈화 이후의 파이프라인이 기존 실험 결과와 일치(Consistency)하는지 최종 점검

# Blocker (이슈 사항)
- [발생] F1-Score 수치 오류 (1.0 초과)
  - 원인: Stem(어간) 변형 처리 과정에서 동일 객체가 중복 카운팅되어 Precision/Recall 계산 시 분자가 분모보다 커지는 현상 발견
  - 해결: 중복 카운팅 로직 수정 및 데이터 전처리 파이프라인 보정 작업 중
